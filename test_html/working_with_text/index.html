Paso a paso

Bueno, esto es un párrafo. Voy a poner un poco de texto de prueba: Lorem ipsum etc. Por acá está lloviendo.

Este es otro párrafo. strong text

Ahora paró de llover (en cursivita).

strong es para indicar la importancia de una palabra/sección.

em (emphasis) es para enfatizar una palabra/sección.

i se usa para transmitir una sensación distinta a una palabra/sección (usado para palabras extranjeras o técnicas).

b es para dirigir la atención a una palabra/sección sin otorgarle mayor importancia.
